redis操作浅谈

介绍：
Redis 开源高性能的key-value数据库，可以将内存中的数据保持在磁盘中，重启的时候可以再次加载进行使用。
读的速度是110000次/s,写的速度是81000次/s。
redis支持string（字符串），hash（哈希），list（列表），set（集合）及zset(sorted set：有序集合)
这里不做安装介绍，有需要的可自行搜索教程。
Redis 客户端语法：redis-cli
